Web Storage Tutorial

Web Storage is a web API that allows to store key/value pairs in the browser.

It is used to store data that is not visible to the user, such as user login credentials, shopping cart contents, and other data that is not meant to be visible to the user.

We could also use cookies to store data that is not meant to be visible to the user. But localStorage is better for this purpose.

If user has disabled third party cookies, then it wont work. In that case, --> localStorage will work.

localStorage is a property of the window object.
localStorage is a collection of key/value pairs.
Save the data in the browser using localStorage.setItem method.
Retrieve the data from the browser using localStorage.getItem method.
Remove the data from the browser using localStorage.removeItem method.
Clear the data from the browser using localStorage.clear method.
