GeoLocation Tutorial

GeoLocation is a web API that allows you to get the location of the user.

User will be prompted to allow the browser to access their location. If the user allows, the browser will get the location of the user. If the user does not allow, the browser will not get the location of the user.

Geolocation is a read-only property of the navigator object.

There are multiple ways to access this location information:

But we will use here getCurrentPosition(): which retrives the user's current position.

We need one callback function to handle the response. The callback function will be called when the response is a success. The callback function will have two arguments:

  • position - This is the object that contains the location information.
  • status - This is the status of the response.

geolocation properties has the following properties:

  • coords - This is the object that contains the location information.
  • timestamp - This is the time at which the position was determined.

We use these properties to get the location information.
